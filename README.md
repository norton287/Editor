# Editor
Simple RTF Editor written in C#
## Getting Started
I used Visual Studio Community 2017 and .NET Framework 4 to build this app.
### Prerequisites
```
.NET Framework 4.7.6
```
### Installing
```
Unzip to C:\Editor and run release app
```
